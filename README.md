<h2>Nigeria Covid-19 Report Dashboard</h2>
<p>This project was developed with Flask version 1.1.2 and Python 3.6.9</p>

<h3>Prerequisite:</h3>
-Install numpy</br>
$ pip install numpy

-Install folium </br>
$ pip install folium

-Install pandas</br>
$ pip install pandas

-Install mysql-connector to connect to mysql database </br>
$ pip install mysql-connector
